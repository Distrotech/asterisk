QUEUE_WAITING_COUNT()

Synopsis

Count number of calls currently waiting in a queue.

Description

Returns the number of callers currently waiting in the specified queuename.

Syntax

QUEUE_WAITING_COUNT([queuename])
Arguments
  • queuename
