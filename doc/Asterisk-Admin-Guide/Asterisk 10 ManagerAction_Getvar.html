Getvar

Synopsis

Gets a channel variable.

Description

Get the value of a global or local channel variable.

Syntax

Action: Getvar
[ActionID:] <value>
[Channel:] <value>
Variable: <value>
Arguments
  • ActionID - ActionID for this transaction. Will be returned.
  • Channel - Channel to read variable from.
  • Variable - Variable name.
